# IT News Digest

AI・IT分野の最新情報を自動収集・整理するリポジトリ。

## フォルダ構成

```
it-news-digest/
├── daily/          # 日次まとめ（YYYY-MM-DD.md）
└── topics/         # トピック別の深掘り記事
```

## 収集カテゴリ

毎サイクルで以下から2〜3カテゴリを選んで収集する：

- **AI Models** — LLM新モデル・ベンチマーク・API変更
- **AI Tools** — 開発者向けツール・IDE統合・エージェント
- **Cloud & Infra** — AWS/GCP/Azure新機能・コスト動向
- **Security** — 脆弱性・PQC・ゼロトラスト
- **Open Source** — 注目リポジトリ・フレームワーク更新
- **Startups** — 資金調達・買収・新サービスローンチ
- **Japan IT** — 国内IT政策・DX事例・スタートアップ

## 更新頻度

270秒ごとに自動サイクル実行（/loop で起動）

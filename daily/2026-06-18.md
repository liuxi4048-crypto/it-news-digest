## 🛠️ AI Developer Tools — 開発ツール・エージェント

### Microsoft Build 2026（主要発表）
- **Windows AIエージェント統合**: WindowsをAIエージェントのネイティブプラットフォームに位置づけ。ローカルデバイス・クラウド・エンタープライズ横断エージェントが対象
- **Microsoft Execution Containers (MXC)**: エージェント向けポリシー駆動の実行レイヤー（早期プレビュー）。WSL対応
- **Agent 365 SDK GA**: 開発ワークフロー内にセキュリティコントロールを直接組み込み可能に
- **Microsoft Foundry**: Build（フレームワーク）/ Deploy（ホスティング）/ Operate（トレース・評価）の3層構成

- **2026年はエージェント元年**: クラウド・エンタープライズ・コンサル・開発ツール全方位で「エージェント化」が共通戦略に

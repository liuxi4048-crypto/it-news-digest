| プロバイダ | 動き | 詳細 |
|-----------|------|------|

## 🔐 Security — 脆弱性・インシデント

### Microsoft Patch Tuesday（2026年6月） — 史上最多規模
- **200件の脆弱性**を一度に修正（Microsoft 史上最多のパッチ密度）
- うち **33件がクリティカル**、**6件がゼロデイ**（うち5件はパッチ前に公開済み）

### 注目 CVE・インシデント
| 種別 | 概要 |
|------|------|
| **CVE-2026-11645** | Chrome V8 エンジンの境界外メモリアクセス。Google が野放し悪用を確認済み |
| **JetBrains Marketplace マルウェア** | 15本の悪意あるプラグインが AI プロバイダの API キーを窃取するキャンペーン |
| **VRChat データ漏洩** | 5/10〜5/12 にクラウド環境への不正アクセス。**240万ユーザー**のプロフィール・ログイン情報が影響 |
| **ServiceNow 認証バイパス（6/9公開）** | Scripted REST API のエンドポイントに未認証でアクセス可能な欠陥。エンタープライズ顧客データが対象 |

### CISA 緊急対応
- LiteSpeed cPanel Plugin の脆弱性を「既知悪用脆弱性カタログ」に追加
- 連邦機関に **2026-06-18 までの修正適用**を義務付け
